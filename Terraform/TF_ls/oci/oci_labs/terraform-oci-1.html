Terraform on OCI

Training Info

The certified trainers for this course are as follows:

  • Andrei Ciorba

Course Description

This is an introductory course which covers the foundational knowledge of virtual infrastructure and Hashicorp’s Terraform toolset for infrastructure automation. The training will cover multiple cloud providers, starting with AWS for the basic examples and then proceeding with OCI, with a wide array of examples.

What Will You Learn

The course is focused on learning Terraform’s concepts and syntax and on applying this knowledge to create and update working configurations. Further expansion of a virtual deployment is covered through the use of functions, provisioners and data sources. Moving forward with more advanced features, the course covers methods of writing more efficient code and working with multiple environments, by introducing variables, modules and managing state files.

Prerequisites

  • basic understanding of any programming or scripting language concepts (functions, variables, etc)
  • basic understanding of simple cloud infrastructure elements (virtual networks, virtual instances, etc)

Target Audience

This course is targeting both developers who want to automate their code testing infrastructure as well as cloud or physical infrastructure-related positions that benefit from automation.
